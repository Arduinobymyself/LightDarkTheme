# LightDarkTheme
Estudo sobre tema tipo Light/Dark usando apenas HTML/CSS e javascript

- Usa CSS para estilizar a página em dois temas fundamentais Light e Dark;
- Além disso usa javascript para dar o dinamismo na troca emtre os temas;
- Usa o bootstrap icons como parte do estilo dos icones sol (light) e lua (dark)
- Usa o local storage para armazenas a preferência de tema selecionada pelo usuário, assim quando deixar a página, ao voltar o tema previamente selecionado será apresentado
